# Select文のソートキーを複数指定する

これは正解できた。

```
select
distinct
sum(price) as count_products_types,
producer
from products
group by price,producer
order by count_products_types DESC, producer ASC
```

order by による並べ替えは2種類のカラムを指定して実行することができる。
2つのカラムをキーにして並べ替えを実行するとどうなるのか？
